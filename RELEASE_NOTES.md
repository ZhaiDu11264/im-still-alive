# im-still-alive v1.0.0

发布日期: 2025-12-24

## 🎯 项目简介

一个简洁优雅的心情打卡应用

## ✨ 主要功能

- 心情打卡系统
- 实时聊天功能
- 广场分享
- 成就系统
- PWA支持
- HTTPS支持
- 响应式设计

## 📋 系统要求

- Node.js 16+
- MySQL 8.0+
- Python 3.8+ (可选，用于SSL证书生成)

## 🚀 快速开始

1. 克隆项目
```bash
git clone https://github.com/your-username/im-still-alive.git
cd im-still-alive
```

2. 自动部署
```bash
npm run setup
```

3. 启动服务
```bash
npm start
```

## 📱 访问地址

- HTTP: http://localhost:3002
- HTTPS: https://localhost:3443
